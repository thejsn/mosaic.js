Basic usage

First, create an element to hold the mosaic.

<span id="imagemosaic"></span>

Then either setup with javascript:

var mosaic = new Mosaic(document.getElementById('imagemosaic'));

// Set dimensions etc.
mosaic.setSize(400, 400, 10, 10);
mosaic.colorBlending = 0.5;

// Add image
mosaic.addSourceFromURL('images/1.jpg');

// add more images...
// mosaic.addSourceFromURL('images/n.jpg');

// Set a target image
mosaic.setTargetFromURL('images/1.jpg');
